# 生成 csr

首先去 csr 在线生成网站open in new window生成自己的 csr

  • 注意：填写所有的信息是不能包含中文和特殊字符 下载下来会有 2 个文件，csr 和 key 文件

# 申请免费证书

免费证书申请网站open in new window 在阿里云域名设置里添加记录

然后在 freessl 网站进行验证，然后下载

full_chain.pem 是 freessl 签发的证书，key.key 是第一步 csr 生成网站下载下来的。 将这两个文件上传到服务器
