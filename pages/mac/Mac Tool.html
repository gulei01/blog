###切换版本

$ pyenv global 3.5.5 # 全局切换
$ python -V # 验证一下是否切换成功
1
2
  • 此处用也可用 local，只对当前目录生效
$ pyenv local 3.5.5 # 当前目录及其目录切换
$ python -V # 验证一下是否切换成功
1
2

  • 此处有坑！

  • 用 pyenv versions 查看，明明已经切换成功，但是用 python -V 却还是系统版本。原因是 pyenv 没有加到$PATH 环境变量里去，解决办法如下：

export PYENV_ROOT=~/.pyenv
export PATH=$PYENV_ROOT/shims:$PATH
1
2

  • 此时，再次查看 python -V，已经成功切换到 pyenv 指定的版本。

  • 可以把上边两句加到~/.bash_profile 里去，永久生效。

  • 有时设置了 pyenv local 版本后，再设置 global 会发现没有生效，可以尝试：

pyenv local --unset
1

  • 解除 local 设置。

  • 要切换回系统版本，用同样命令：

$ pyevn global system
1

# Mac 非苹果鼠标蓝牙连接不到解决办法

非苹果鼠标连接其他设备以后，在这台苹果电脑就不会自动连接

sudo rm /Library/Preferences/com.apple.Bluetooth.plist
1

删除只会重新连接的时候

按 空格 进行连接
